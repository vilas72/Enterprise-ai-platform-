## How to Start Development

```bash
# Activate environment
source .venv/bin/activate  # Linux/Mac
.venv\Scripts\Activate.ps1  # Windows

# Start API server
uvicorn main:app --reload

# Run tests (after installing pytest)
pytest tests/ -v

# Check code quality
black app/ --check
flake8 app/
```

## Known Limitations

1. **In-Memory Storage**: Conversations stored in RAM, lost on restart
2. **Single-Instance**: No horizontal scaling yet
3. **No Persistence**: Vector embeddings not persisted
4. **No Auth**: API endpoints accessible without authentication
5. **Mock Search**: Keyword search works on 100% data, not production-optimized

All limitations addressed in Phases 2-3 of roadmap.
